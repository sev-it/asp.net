Перед запуском хоста, в каталоге WebFileBrowser\FileBrowser.Web\app\js, в файле services.js необъодимо изменить домен и порт, на те, под которыми развернут back-end (FileBrowser.Api)
Для запуска хоста необходимо перейти в корневой каталог приложения:
cd WebFileBrowser\FileBrowser.Web
И запустить хост:
npm start
